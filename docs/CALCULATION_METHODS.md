# 계산 방법

## 1. 데이터 단위

계산 주기는 월입니다. 자산 `i`의 `t`월 수익률을 `r_{i,t}`, 목표 비중을 `w_i`로 둡니다.

가격 CSV를 사용하면 각 월의 가장 늦은 날짜 수정주가 `P_t`로 월 수익률을 계산합니다.

```text
r_t = P_t / P_(t-1) - 1
```

## 2. 적립식 현금흐름

- 월초 적립: 적립금에서 거래비용을 차감한 뒤 목표비중으로 매수하고 해당 월 수익률을 적용합니다.
- 월말 적립: 해당 월 수익률을 먼저 적용한 뒤 적립금에서 거래비용을 차감하고 목표비중으로 매수합니다.
- 납입 원금은 거래비용 차감 전 현금흐름의 합계입니다.

## 3. 리밸런싱과 거래비용

리밸런싱 직전 자산금액을 `H_i`, 포트폴리오 금액을 `V`, 목표금액을 `T_i = V w_i`라 하면 회전율 금액은 다음과 같습니다.

```text
turnover = 0.5 × Σ |T_i - H_i|
rebalance_cost = turnover × cost_rate
```

`cost_rate = 입력 bp / 10,000`입니다. 비용을 차감한 잔액을 목표비중으로 재배분합니다.

## 4. 시간가중수익률(TWRR)

각 월의 외부 현금흐름 전후를 분리해 월 수익률 `R_t`를 구하고 단위지수를 누적합니다.

```text
unit_index_T = 100 × Π(1 + R_t)
TWRR_annual = [Π(1 + R_t)]^(12/N) - 1
```

따라서 월 적립금 규모와 납입 시점이 TWRR 자체를 왜곡하지 않습니다.

## 5. 금액가중수익률(MWRR)

초기 투자금과 월 적립금을 음의 현금흐름, 최종 자산을 양의 현금흐름으로 놓고 월 내부수익률 `q`를 이분법으로 풉니다.

```text
0 = Σ CF_t / (1 + q)^t
MWRR_annual = (1 + q)^12 - 1
```

현금흐름에 따라 해가 존재하지 않거나 복수일 수 있으므로 화면에서 `—`로 표시될 수 있습니다.

## 6. 변동성·샤프·소르티노

월 수익률 표준편차를 `σ_m`, 연 무위험수익률을 `r_f`라 하면:

```text
volatility_annual = σ_m × sqrt(12)
rf_monthly = (1 + r_f)^(1/12) - 1
Sharpe = mean(R_t - rf_monthly) / stdev(R_t) × sqrt(12)
```

소르티노는 `R_t - rf_monthly` 중 음수 부분만 사용한 하방편차를 연율화하고, 연환산 수익률과 연 무위험수익률의 차이를 나눕니다.

## 7. 최대낙폭(MDD)

적립금으로 인한 자산 증가를 제외하기 위해 TWRR 단위지수에서 계산합니다.

```text
drawdown_t = unit_index_t / max(unit_index_0 ... unit_index_t) - 1
MDD = min(drawdown_t)
```

낙폭 비교 차트는 포트폴리오와 선택한 벤치마크의 TWRR 단위지수에 위 공식을 각각 적용합니다. 과거 낙폭 순위표는 전고점 다음 월을 `시작`, 해당 구간의 최저점을 `저점`, 전고점 이상으로 돌아온 최초 월을 `전고점 회복`으로 정의합니다. `수중 기간`은 시작월부터 회복월까지이며, 미회복 구간은 분석 종료월까지 계산합니다.

## 8. 베타와 상관관계

```text
beta = cov(portfolio, benchmark) / var(benchmark)
correlation(i,j) = cov(i,j) / [stdev(i) × stdev(j)]
```

모두 선택 기간의 월 수익률을 사용합니다.

## 9. 위험기여도

월 공분산행렬을 `Σ`, 비중 벡터를 `w`라 하면 자산 `i`의 분산 기여 비율은 다음과 같습니다.

```text
RC_i = w_i × (Σw)_i / (w'Σw)
```

음의 상관관계가 큰 자산은 위험기여도가 음수가 될 수 있습니다.

## 10. 실질 최종자산

입력한 연 물가상승률 `π`, 기간 `Y`년을 사용합니다.

```text
real_final = nominal_final / (1 + π)^Y
```

## 11. 몬테카를로

### 부트스트랩

백테스트에서 얻은 월 포트폴리오 수익률을 매월 복원추출합니다. 과거 분포의 비정규성과 꼬리를 일부 보존하지만, 월간 자기상관과 시장 국면의 지속성은 보존하지 않습니다.

### 정규모형

과거 월 수익률의 평균과 표준편차를 모수로 정규 난수를 생성하며, 월 수익률은 MVP 안전장치로 `-75%`에서 `+75%` 사이로 제한합니다.

각 경로에서 월 적립 → 수익률 적용 → 월 인출 순으로 계산합니다. 최종값의 P10, P25, P50, P75, P90과 목표 달성·자산 고갈 비율을 표시합니다.

## 12. 해석상 주의

- 과거 수익률 분포가 미래에 반복된다는 보장이 없습니다.
- 월간 독립추출은 변동성 군집, 장기 평균회귀, 금리·환율 국면을 반영하지 않습니다.
- 실제 세금, 슬리피지, 호가 스프레드, 환전비용, 펀드 추적차이, 거래정지·상장폐지는 별도 모형이 필요합니다.
- 여러 전략을 반복 탐색하면 데이터 스누핑과 과최적화가 발생합니다.
